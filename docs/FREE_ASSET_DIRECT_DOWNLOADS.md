## Required verification after transfer

For every received archive:

1. Verify the file is non-empty.
2. Verify it opens as a ZIP archive.
3. Record byte size and SHA-256.
4. List all files and reject executables or unexpected formats.
5. Confirm the embedded license/readme matches the source page.
6. Extract into a temporary intake directory only.
7. Generate a contact sheet.
8. Classify each item as approved, experimental, or rejected.
9. Import only selected files into canonical project paths in a separate PR.
