# ChatGPT Asset Generation Backlog

## Purpose

Define which missing assets should be generated by ChatGPT when suitable free commercial-use assets cannot be found or imported.

This document does not add runtime assets and does not modify game code.

## Locked visual direction

- Stylized 3D fantasy mobile-game look
- Hand-painted texture feel
- Clean silhouette at small size
- Soft cinematic lighting
- Transparent background where applicable
- No text, watermark, frame, or baked UI label unless the asset is explicitly a UI frame
- Avoid pixel art, flat clip-art, photorealism, and side-view sprite styles

## Generation workflow

For every batch:

1. Generate no more than 4 related assets at once.
2. Review style consistency before continuing.
3. Reject any result with inconsistent camera, outline, lighting, or padding.
4. Preserve the approved prompt and final filename mapping.
5. Add only approved PNGs to the repository in a separate asset PR.
