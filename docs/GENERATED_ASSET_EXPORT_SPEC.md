# Generated Asset Export Specification

## Goal

Provide one consistent technical standard for PNG assets generated by ChatGPT before Coco imports them into the repository.

## General PNG rules

- Format: PNG
- Colour: sRGB
- Background: transparent unless explicitly stated
- No watermark
- No baked labels or text
- No unnecessary empty canvas
- Preserve enough transparent padding to avoid clipping glow, weapons, ears, horns, or VFX

## UI frames

- Master size: 1024×1024 or larger
- Corners and edge ornaments must support 9-slice extraction
- Centre region should be visually quiet and stretchable
- Do not bake shadows that extend excessively outside the frame
- Keep border thickness consistent across related assets

Proposed destination:

`assets/ui/generated/frames/`

## UI icons

- Master size: 256×256
- Important silhouette must remain readable at 24×24
- Central object should occupy roughly 72–82% of the canvas
- Keep lighting direction consistent across the set
- Avoid tiny decorative details that disappear on mobile

Proposed destination:

`assets/ui/generated/icons/`

## Character and enemy cutouts

- Master size: 1024×1024
- Transparent background
- Full body visible
- 3/4 front view
- Feet must not be clipped
- Subject centred with modest side padding
- Head and weapon silhouette must remain readable after reduction
- Avoid ground plane and cast shadow unless separately requested

Proposed destinations:

- `assets/characters/generated/`
- `assets/monsters/generated/`
- `assets/bosses/generated/`

## VFX

Single-frame VFX:

- 512×512 or 1024×1024
- Transparent background
- Effect centred
- Soft edge alpha

Sprite sheets:

- Fixed-size cells
- No spacing between cells unless documented
- Frame order left-to-right, then top-to-bottom
- Include frame count, columns, rows, FPS suggestion, and loop/one-shot flag in the manifest

Proposed destination:

`assets/vfx/generated/`

## Naming

Use lowercase snake_case.

Examples:

- `ui_panel_frame_primary.png`
- `icon_gold.png`
- `class_fighter.png`
- `monster_slime.png`
- `vfx_shield_aura_sheet.png`

Do not use:

- spaces
- Thai filenames
- version words such as final, latest, new, fixed
- opaque generated IDs

## Validation before import

Coco must verify:

- file is not zero bytes
- PNG decodes successfully
- dimensions match the declared manifest
- alpha channel exists when required
- no duplicate destination path
- no duplicate logical ID
- filename follows naming rules
- source is recorded as `generated_by_chatgpt`
- approval status is recorded

## Runtime boundary

Importing a PNG does not authorize runtime integration. Any change to Three.js materials, sprites, animation timing, combat effects, camera, board, or game state requires a separate scoped task.
