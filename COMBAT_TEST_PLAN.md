# Combat Step 1 — Acceptance Test Plan

เป้าหมายของขั้นตอนนี้คือทำให้ระบบต่อสู้ของ Map 1 เสถียรพอสำหรับนำระบบสกิล ไอเทม และบาลานซ์มาต่อยอด

## กติกาที่ล็อกแล้ว

- Beast Lord โจมตีระยะกลาง 3 ช่อง
- การเดินใช้ 4 ทิศ แต่ระยะโจมตีวัดแบบ Chebyshev จึงตีทแยงได้
- ยูนิตล็อกเป้าหมายและตรวจใหม่ทุก 0.4 วินาที
- ตัวประชิดและตัวยิงต้องหา “ช่องโจมตีที่ว่างและเดินถึงได้”
- Tank/แนวหน้ามี Threat สูงกว่า
- Tank ใช้ Taunt ระยะ 3 ช่อง นาน 3 วินาที คูลดาวน์ 8 วินาที และลดดาเมจที่ได้รับ 35% ระหว่าง Taunt
- Shadow Assassin วาร์ปครั้งเดียวตอนเริ่มไฟต์ไปช่องว่างติดแนวหลัง โดยให้ความสำคัญกับตัวระยะไกลก่อน
- ตัวยิงไกลไม่ถอยเมื่อถูกประชิด และไม่มีระยะขั้นต่ำ
- Projectile ตามเป้าหมายและทำดาเมจเมื่อถึง
- ไม่มี Line of Sight: ยูนิตและสิ่งกีดขวางไม่บล็อก Projectile
- ถ้าเป้าหมายตายก่อน Projectile ถึง ให้ Projectile หาย
- Projectile ที่ยิงออกไปแล้วไม่หายเพียงเพราะผู้ยิงตาย
- เมื่อจบด่านหรือเริ่ม Retry ต้องล้าง Target, Taunt, Warp state และ Projectile

## กรณีทดสอบบังคับ

### 1. Melee 1 ต่อ 1
- ยูนิตเดินเข้าหาเป้าหมายได้
- หยุดในระยะและโจมตีได้
- ตีทแยงได้
- ไม่มีการเดินเข้าไทล์เดียวกับศัตรู

### 2. Melee หลายตัวรุมเป้าหมาย
- อย่างน้อย 4 ตัวสามารถกระจายรอบเป้าหมายได้
- ตัวที่ยังเข้าไม่ถึงต้องรอหรือเลือกเส้นทางใหม่โดยไม่สั่นเป้าหมายทุกเฟรม
- ไม่มีสองตัวครองช่องเดียวกัน

### 3. เป้าหมายถูกล้อม
- ถ้าไม่มีช่องยิงหรือช่องประชิดที่เข้าถึงได้ ต้องเลือกเป้าหมายอื่น
- การต่อสู้ต้องไม่ค้างถาวร

### 4. แนวหน้า Threat และ Taunt
- ศัตรูทั่วไปควรเลือก Blade Master/Beast Lord ก่อนตัวระยะไกลเมื่อระยะเส้นทางใกล้เคียงกัน
- Tank ที่ใช้ Taunt ดึงเฉพาะศัตรูในระยะ 3 ช่อง
- ศัตรูที่โดน Taunt เปลี่ยน Target มาหา Tank ทันที
- ระหว่าง Taunt Tank รับดาเมจน้อยลงประมาณ 35%
- เมื่อครบ 3 วินาที ศัตรูกลับเข้าสู่ระบบเลือกเป้าหมายปกติ
- Tank ไม่ใช้ Taunt ซ้ำก่อนครบคูลดาวน์ 8 วินาที
- ถ้า Tank ตายระหว่าง Taunt ศัตรูต้องปลด Taunt และหาเป้าหมายใหม่
- Tank ที่อยู่ไกลเกินไปไม่ควรดึงศัตรูข้ามครึ่งกระดาน

### 5. Shadow Assassin Opening Warp
- ตอนเริ่มไฟต์ Assassin วาร์ปครั้งเดียวโดยไม่เดินผ่านแนวหน้า
- ให้ความสำคัญกับตัวระยะไกลและแนวหลัง
- วาร์ปลงเฉพาะไทล์ว่างติดกับเป้าหมาย
- ถ้าช่องที่ต้องการเต็ม ให้ลองช่องข้างเคียงอื่น
- ถ้าไม่มีช่องว่างรอบแนวหลังเลย ให้กลับไปใช้การเดินปกติ
- หลังวาร์ปต้องล็อกเป้าหมายที่เลือกไว้ก่อน
- Retry ด่านเดิมแล้ว Assassin ต้องวาร์ปใหม่ได้หนึ่งครั้ง

### 6. Ranged Positioning
- ตัวยิงหาไทล์ว่างภายในระยะยิงแทนการเดินไปยังไทล์ศัตรู
- Beast Lord หยุดเมื่ออยู่ในระยะ 3 ช่อง
- Sniper และ Mage ไม่เดินประชิดโดยไม่จำเป็น
- เมื่อถูกประชิด ตัวยิงต้องยืนยิงสู้ ไม่ถอยและไม่ Kite

### 7. Projectile และแนวยิง
- ดาเมจเกิดเมื่อกระสุนถึงเป้าหมาย
- เป้าหมายตายก่อนถึง กระสุนหาย
- ผู้ยิงตายหลังยิง กระสุนที่ปล่อยแล้วเดินทางต่อ
- Projectile ยิงผ่านยูนิตฝ่ายเดียวกันและฝ่ายตรงข้ามได้
- สิ่งกีดขวางไม่บล็อก Projectile
- Pause ต้องหยุด Projectile
- Speed x2 ต้องเพิ่มความเร็วการจำลองอย่างสอดคล้อง

### 8. Reset และ Retry
- เมื่อจบด่าน Projectile ทั้งหมดหาย
- Target, blocked timer, Taunt, Warp state และ animation state ถูกรีเซ็ต
- แพ้แล้ว Retry ด่านเดิมได้โดยไม่มีเป้าหมายเก่าค้าง

### 9. Full Run
- เล่นตั้งแต่ด่าน 1 ถึง 15 โดยไม่มี JavaScript error
- ไม่มีการต่อสู้ค้างเกินเวลาที่กำหนด
- ผลการจัดแนวหน้า/หลังต้องสม่ำเสมอกว่าระบบเดิม
- Assassin ไม่ทำให้แนวหลังถูกล้างแบบไร้ทางตอบโต้ทุกครั้ง
- Tank ไม่ถึกหรือดึงศัตรูมากจนกลายเป็นตัวเลือกบังคับเพียงแบบเดียว

## เกณฑ์ผ่านขั้นตอนที่ 1

ขั้นตอนที่ 1 ถือว่าผ่านเมื่อ:

1. กรณีทดสอบ 1–8 ผ่านทั้งหมด
2. Full Run อย่างน้อย 3 รอบไม่มี error หรือ soft lock
3. ทีมและตำแหน่งเดียวกันไม่ควรมีผลต่างรุนแรงแบบชนะรวดกับแพ้ตั้งแต่ต้นโดยไม่มีเหตุจาก RNG ที่อธิบายได้
4. ไม่มีบั๊ก Blocker ที่ทำให้เล่น Map 1 ต่อไม่ได้
